# Görseller Klasörü

Bu klasöre kendi tasarladığınız görselleri ekleyin:

## Cookie Görselleri:
- `cookie_torii.png` - Torii kapısı cookie (🏯)
- `cookie_halloween.png` - Halloween cookie (🎃)
- `cookie_ramazan.png` - Ramazan cookie
- `cookie_noel.png` - Noel cookie
- `cookie_anne.png` - Anne cookie
- `crack_left.png` - Kırılma efekti (sol)
- `crack_right.png` - Kırılma efekti (sağ)
- `sparkle.png` - Parlama efekti

## Kullanım:
1. Görselleri bu klasöre kopyalayın
2. `pubspec.yaml` dosyası zaten yapılandırılmış
3. Kodda `Image.asset('assets/images/cookie_torii.png')` şeklinde kullanın
